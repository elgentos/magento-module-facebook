# Facebook Pixel

Magento 1 module which enables all Facebook Pixel options.


## Usage

1. In admin go to : System -> Configuration -> General -> Facebook Pixel
2. Add your Pixel ID & Save

Magento 1 module which enables all Facebook Pixel options.

## Tracking Options


* View content
* Search
* Add to cart
* Add to wishlist
* Initiate checkout
* Add payment info
* Make purchase
* Lead
* Complete registration
